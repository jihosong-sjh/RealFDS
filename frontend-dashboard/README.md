# Frontend Dashboard (프론트엔드 대시보드)

실시간 금융 거래 탐지 시스템(RealFDS)의 웹 기반 사용자 인터페이스입니다.

## 서비스 목적

- **실시간 알림 표시**: WebSocket을 통해 탐지된 의심스러운 거래 알림을 즉시 표시
- **연결 상태 모니터링**: WebSocket 연결 상태를 실시간으로 표시
- **알림 히스토리**: 최근 100개의 알림 히스토리 유지 및 표시

## 기술 스택

- **React 18**: 컴포넌트 기반 UI 라이브러리
- **TypeScript 5**: 타입 안전성을 위한 정적 타입 체크
- **Vite 5**: 빠른 개발 서버 및 빌드 도구
- **WebSocket API**: 실시간 양방향 통신
